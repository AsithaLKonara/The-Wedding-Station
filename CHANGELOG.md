# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of The Wedding Station portfolio website
- Next.js 14 with TypeScript and App Router
- Responsive homepage with Hero, Gallery, Video Reel, About, and Contact sections
- Facebook Graph API integration for automatic post syncing
- Server-side caching with Redis and in-memory fallback
- Contact form with SendGrid and SMTP support
- Lightbox component with keyboard navigation
- Video player with thumbnails
- Comprehensive test suite with Jest and React Testing Library
- CI/CD pipeline with GitHub Actions
- Full TypeScript type definitions
- ESLint and Prettier configuration
- Husky pre-commit hooks
- Comprehensive README with setup and troubleshooting guides

### Features
- Automatic Facebook post fetching and caching
- Masonry-style photo gallery
- Responsive video reel section
- Server-side email sending
- Accessibility features (ARIA labels, keyboard navigation)
- Performance optimizations (lazy loading, ISR, image optimization)
- Dark mode support (optional)
- Smooth animations with Framer Motion

### Technical
- Next.js 14.2.0
- React 18.3.0
- TypeScript 5.3.0
- Tailwind CSS 3.4.0
- Framer Motion 11.0.0
- Jest 29.7.0
- React Testing Library 14.1.2
